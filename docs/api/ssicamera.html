ssi_sensor_CameraScreen
  CameraScreen  SENSOR
 
           Captures the screen or parts of the screen and provides the grabbed frames as a video stream.
 
             +  fps   DOUBLE   1   '2.500000e+001'   frames per second 
             +  full   BOOL   1   'true'   full screen toggle (if true region is ignored) 
             +  region   CHAR   1024   '0,0,0,0'   region (left, top, width, height) 
             +  mouse   BOOL   1   'false'   capture mouse cursor 
             +  mouse_size   UINT   1   '4'   size of mouse cursor in pixel 
             +  resize   BOOL   1   'false'   resize image to widthxheight (requires ssiimage[d].dll!) 
             +  resize_width   UINT   1   '0'   resize width in pixels 
             +  resize_height   UINT   1   '0'   resize height in pixels 
             +  resize_method   INT   1   '1'   resize interpolation method (0=NN,1=LINEAR,2=CUBIC,3=AREA) 
 
           Available channels:
 
             +  video   Delivers a stream of uncompressed images in the Device Independent Bitmap (DIB) file format of the desired sub type.
